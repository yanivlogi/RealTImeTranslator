# Translate Screen

תיאור
- סקריפט Python שמבצע OCR בזמן אמת על אזור מסך מוגדר ומציג תרגום בעברית בחלון עליון.
- מבוסס על `easyocr` לזיהוי טקסט ו-`deep_translator` (GoogleTranslator) לתרגום.

קבצים
- הקובץ הראשי: [translate.py](translate.py)

תלותיות
- Python 3.8+
- חבילות pip:
  - `easyocr`
  - `opencv-python`
  - `numpy`
  - `Pillow`
  - `deep-translator`
- `tkinter` (מגיעה כברירת מחדל ברוב התקנות Python על Windows)
- שימו לב: `easyocr` דורשת `torch` — התקינו `torch` לפי ההנחיות של PyTorch למערכת שלכם אם רוצים GPU.

התקנה
```bash
python -m pip install easyocr opencv-python numpy Pillow deep-translator
# התקנת torch לפי מערכת/גרסה אם רוצים אפשרות GPU
```

שימוש
1. התאימו את אזור הסריקה על ידי עריכת משתנה `bbox` בראש הקובץ `translate.py`.
   - פורמט: `[x1, y1, x2, y2]` (פיקסלים, קצה שמאלי עליון ואז קצה ימני תחתון).
2. הריצו:
```bash
python translate.py
```
3. יפתחו שלושה חלונות:
   - מסגרת אדומה (חלון ראשי ללא כותרת) המייצגת את אזור הסריקה.
   - חלון תרגום שחור שמציג את הטקסט המתורגם.
   - חלון בקרה (מרכז שליטה) לשינוי מיקום והצגה/הסתרה של המסגרת והחלון.

פקדי חלון הבקרה
- "הצג/הסתר מסגרת אדומה": מפעיל/מכבה את המסגרת האדומה.
- לחצנים עם חצים: להזזת אזור הסריקה או המיקום של חלון התרגום ב-10 פיקסלים לכל לחיצה.

התאמות ושינויים נפוצים
- לשנות שפות OCR: ערכו את שורת ההתחלה `reader = easyocr.Reader(['en'], gpu=False)` — החליפו `'en'` לשפה הרצויה.
- להפעיל GPU עבור `easyocr`: העבירו `gpu=True` והתקינו `torch` תואם.
- פילוח/סינון טקסט נעשה בקוד לפני שליחת הטקסט לתרגום — ישנן בדיקות על אורך והסרת תווים מיוחדים.

הערות ובעיות ידועות
- הקוד מדלג על חריגות באופן גורף (`except: pass`) — אם משהו נכשל, ייתכן שלא תראו הודעת שגיאה מפורשת.
- `ImageGrab` עשוי לדרוש הרשאות צילום מסך ב-Windows במערכות מסוימות.
- איכות ה-OCR תלויה בבהירות ובניגודיות של הטקסט באיזור הסריקה.

שיפורים אפשריים
- הוספת לוג או הודעות שגיאה מפורטות במקום `except: pass` כדי לאתר בעיות.
- תמיכה בהרבה שפות OCR במקביל.
- שימור היסטוריית תרגום או העתקה אוטומטית ללוח.

אם תרצה, אני יכול:
- להפוך את `bbox` להגדרה ניתנת לשינוי דרך ממשק (לדוגמה חיצים במקלדת).
- להוסיף קובץ `requirements.txt` וליצור הוראות התקנה מדויקות יותר.
